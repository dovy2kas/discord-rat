# Discord RAT

This is a remote access tool (RAT) that uses discord as a C2.

## Features

- Keylogger (!startkeylogger, !dumpkeylogger)
- DDOS (!tcp, !udp, !icmp, !http)
- Sending commands individually and to everyone
